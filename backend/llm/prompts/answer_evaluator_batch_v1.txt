Sua tarefa é avaliar um conjunto de respostas discursivas (Resposta Curta ou Cálculo) comparando cada uma com a resposta esperada.

## Questões e Respostas para Avaliação
Abaixo está o objeto JSON contendo as questões, gabaritos e respostas do aluno:

{{questions_json}}

## Instruções de Avaliação
1. Avalie CADA questão individualmente.
2. Atribua um score de 0 a 100 baseado na rubrica abaixo.
3. Forneça uma justificativa pedagógica concisa para cada nota.
4. Mantenha o `question_id` original para cada item.

## Rubrica de Avaliação
- 100: Resposta completamente correta e bem justificada.
- 75: Quase correta, com pequeno erro ou omissão.
- 50: Parcialmente correta, conceito entendido mas execução incompleta.
- 25: Mínimo de compreensão, conceito errado mas tentativa relevante.
- 0: Resposta errada ou sem relação com a pergunta.

## Formato de Saída (JSON Obrigatório)
Retorne ESTRITAMENTE um objeto JSON com a seguinte estrutura:

{
  "results": [
    {
      "question_id": "uuid-da-questao",
      "score": 85.0,
      "justification": "Explicação do porquê desta nota."
    }
  ]
}

Não adicione textos explicativos fora do JSON.
